# simplegenposts

Генератор постов из мира ИИ для социальных сетей.

Простое веб-приложение для генерации постов на основе URL источника или введенных тезисов с использованием локальных ИИ-моделей (например, через `llama.cpp`, `Ollama`, `LM Studio` и другие серверы, поддерживающие API в формате OpenAI `/v1/chat/completions`).

## Возможности

- **Выбор источника данных**: Возможность ввести URL источника или написать тезисы. Приложение поддерживает одновременное использование обоих полей: материалы из URL дополняются введенными тезисами, и наоборот.
- **Настройка размера поста**: Выбор между подробным и кратким форматом.
- **Настройка характера поста**: Выбор тональности: доступно (понятно для всех), вовлечено, сдержанно, серьезно.
- **Генерация с помощью ИИ**: Использование локальных ИИ-моделей через API endpoints (например, `http://127.0.0.1:8080/v1/chat/completions`).
- **Управление постами**: Возможность редактирования, копирования, удаления и отправки поста в отложенные (свернуть).
- **Отложенные посты**: Список отложенных постов, которые можно развернуть для редактирования, копирования или удаления.
- **Настройки ИИ-модели**: 
  - Адрес сервера (по умолчанию: `http://127.0.0.1:8080`)
  - API KEY (по умолчанию: пусто)
  - Модель (по умолчанию: `Qwen-AgentWorld-35B-A3B-UD-IQ3_S.gguf`)

## Запуск приложения

Приложение является статическим веб-приложением и не требует серверной части. Для запуска достаточно открыть файл `simplegenposts.html` в любом современном браузере:

1. Скачайте репозиторий или скопируйте файл `simplegenposts.html`.
2. Откройте файл `simplegenposts.html` в браузере (двойной клик по файлу или перетаскивание в окно браузера).

## Настройка ИИ-сервера

Приложение предназначено для работы с локальными ИИ-серверами, поддерживающими API в формате OpenAI (эндпоинт `/v1/chat/completions`). 

### Примеры серверов:

- **llama.cpp**: Запустите сервер с поддержкой API:
  ```bash
  llama-server --port 8080 -m your_model.gguf
  ```
- **Ollama**: Убедитесь, что сервер запущен и поддерживает прокси OpenAI, или используйте соответствующий endpoint.
- **LM Studio**: Запустите локальный сервер в LM Studio и укажите адрес и порт в настройках приложения.

### Настройки в приложении:

Нажмите кнопку "Настройки ИИ" в верхнем правом углу приложения, чтобы указать:
- **Адрес**: Адрес вашего ИИ-сервера (по умолчанию: `http://127.0.0.1:8080`).
- **API KEY**: Ключ доступа, если требуется вашим сервером.
- **Модель**: Название вашей ИИ-модели (по умолчанию: `Qwen-AgentWorld-35B-A3B-UD-IQ3_S.gguf`).

## Структура приложения

Приложение состоит из одного файла `simplegenposts.html`, который содержит:
- HTML-разметку для формы ввода данных, настроек поста и управления постами.
- CSS-стили для адаптации под десктопный интерфейс.
- JavaScript-код для логики генерации постов, управления отложенными постами и взаимодействия с ИИ-сервером.
